# Leave Export API - Data Fields Documentation

> [!NOTE]
> This document details the fields available in the Leave Export/Import Excel sheet generated by the Leave Export API. It is designed for HR, administrators, and knowledge-sharing purposes, outlining what each column represents without technical jargon.

## Overview
The **Leave Export API** generates an Excel document (`leaves.xlsx`) containing up-to-date employee leave balances. This file serves a dual purpose: it acts as a **report** of current leave usage and as a **template** that administrators can update and re-import into the system to modify employee leave balances in bulk.

## List of Exported Fields

| Field Name | Description | Required for Import? |
| :--- | :--- | :--- |
| **Action** | Specifies what operation to perform when re-importing the file. Use **'I'** to insert new data or **'U'** to update existing data. | Yes |
| **Employee Code** | The unique identifier or code assigned to the employee (e.g., EMP001). This is used by the system to match the exact employee. | Yes |
| **Employee Name** | The full name of the employee (First Name + Last Name). Included for readability so you know whose record you are viewing. | No (Reference Only) |
| **Total Available Leaves** | The current leave balance (Planned Leaves or PL) available for the employee to use. When updating leave balances, you will typically change this number. | Yes (If Updating) |
| **Leaves Taken (Current Year)** | The sum of all approved leaves the employee has taken across the current calendar year. | No (Reference Only) |
| **Accrued Leaves** | The total number of leaves the employee has officially earned so far in the current year, automatically calculated by the system based on the monthly policy (e.g., 2 leaves per month). | No (Reference Only) |

## How to use the exported file

1. **Download:** Click the export button in the leaves section to download the `leaves.xlsx` file.
2. **Review:** Open the file in Microsoft Excel or Google Sheets to check the current leave balances.
3. **Update:** 
   - To update an employee's balance, set the **Action** column to `U`.
   - Change the number in the **Total Available Leaves** column to the new desired balance. 
4. **Upload:** Save the file and use the **Import Leaves** feature in the system to upload the file. The system will read the spreadsheet and automatically adjust the balances.
